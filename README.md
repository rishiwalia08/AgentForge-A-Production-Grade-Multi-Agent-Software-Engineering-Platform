# Autonomous Multi-Agent Software Engineering Platform

This repository will be built phase-by-phase.

## Phase 1
- Project folder structure
- LangGraph state design
- Minimal FastAPI entrypoint
- Small state-level validation test

## Proposed Structure
- `backend/`: FastAPI app, LangGraph orchestration, and Python services
- `frontend/`: Next.js UI placeholder for later phases
- `docs/`: architecture notes and phase walkthroughs
- `data/`: local artifacts and generated examples
- `memory/`: optional local exports for short-term debugging notes

## Local Model Direction
The project will use Ollama for local development models when we add agents and tools.
